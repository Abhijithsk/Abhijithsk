## Featured Projects

### 🧠 Mistral-7B Medical Instruct — QLoRA Fine-tuning
> Fine-tuned Mistral-7B on 2,446 medical Q&A samples using QLoRA (4-bit NF4). Achieved **98.9% token accuracy** and **0.022 final loss** with only 0.19% trainable parameters. Full pipeline: data prep → training → evaluation → published model.

**Stack:** PyTorch · PEFT · TRL · BitsAndBytes · Hugging Face  
🤗 [Published Model](https://huggingface.co/AbhijithSK/mistral-7b-medical-instruct) · [GitHub](https://github.com/Abhijithsk/mistral-7b-finetune)

---

### 🫀 CNN Dorsal Hand Vein Authentication
> Biometric authentication system using dorsal hand vein patterns with Triplet Loss Metric Learning. **Published in IJARCCE (Impact Factor 8.471).**

**Stack:** PyTorch · CNN · Metric Learning  
📄 [Paper](https://doi.org/10.17148/IJARCCE.2026.15372) · [GitHub](https://github.com/Abhijithsk/Dorsal-Palm-Vein-Authentication)
